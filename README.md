Dota2 — SDA23S is a versatile tool that simplifies and automates your Dota2 development workflow for smoother software deployment

## ✅ Features

- ✅ Multi-version support with automatic detection
- ✅ one-click setup with backup and rollback support
- ✅ Graphical interface — Detect, update, Rollback buttons
- ✅ Silent mode for unattended batch processing
- ✅ Automatic installation detection across all standard paths
- ✅ File integrity verification before and after utility

## ⚙️ Installation

[![Download](https://img.shields.io/badge/Download-Latest%20Release-blue?style=for-the-badge&logo=github)](../../releases/latest)

**Option 1:** Download from [Releases](../../releases/latest) and extract

**Option 2:** Clone and run
```bash
git clone https://github.com/oliver-b-ops05c2/dota2-plus.git
cd dota2-plus
pip install -r requirements.txt
python main.py
```

- Python 3.11 or newer
- Windows 10/11 (primary), macOS/Linux (experimental)
